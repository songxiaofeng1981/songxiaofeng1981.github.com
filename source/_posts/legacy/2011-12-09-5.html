$ sudo apt-get install sun-java6-bin sun-java6-jre sun-java6-jdk

设置默认版本

$ sudo update-java-alternatives -s java-6-sun

 

编辑 /etc/jvm.

$ sudo vi /etc/jvm5

确定java6在最顶端

/usr/lib/jvm/java-6-sun
/usr/lib/jvm/java-gcj
/usr/lib/jvm/ia32-java-1.5.0-sun
/usr/lib/jvm/java-1.5.0-sun
/usr

保存

设置环境变量

$ vi $HOME/.bash_profile

增加下面行

export JAVA_HOME=/usr/lib/jvm/java-6-sun
export PATH=$PATH:$JAVA_HOME/bin

测试你的jdk版本

$ java -version

输入下面：

java version "1.6.0_03"Java(TM) SE Runtime Environment (build 1.6.0_03-b05)Java HotSpot(TM) Server VM (build 1.6.0_03-b05, mixed mode)
编辑helloworld。java 
$ vi HelloWorld.java
代码如下：
publicclass HelloWorld {publicstaticvoid main(String[] args){System.out.println("Hello, World!");    }}

保存 编译执行 
$ javac HelloWorld.java
 $ java  HelloWorld

输出：
Hello, World!
