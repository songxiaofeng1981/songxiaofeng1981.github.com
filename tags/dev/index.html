为Visual Studio code增加markdown自定义代码提示

为Visual Studio code增加markdown 自定义代码提示

文件->首选项->用户代码片段
编辑 markdown.json
文件->首选项->设置-> 增加 "editor.tabCompletion": true

参考资料

代码片段
